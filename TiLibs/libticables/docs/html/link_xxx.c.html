ticables_cable_set_raw

TIEXPORT1 int TICALL ticables_cable_set_raw(CableHandle* handle, int state)

Set the raw cable state (if the hardware supports raw access.) An input of 3 is the default state; 2 means to pull the D0 ("red") line low; 1 means to pull the D1 ("white") line low; 0 means both (never used in the TI link protocol.)

handle :
 a previously allocated handle
state :
 bit mask of lines to pull low
Return value :
 0 if successful, an error code otherwise.

ticables_cable_get_raw

TIEXPORT1 int TICALL ticables_cable_get_raw(CableHandle* handle, int *state)

Get the raw cable state (if the hardware supports raw access.) A bit set in the output means the line is high (1 for the D0 or "red" line, 2 for the D1 or "white" line.)

handle :
 a previously allocated handle
state :
 pointer to variable to store current state
Return value :
 0 if successful, an error code otherwise.
