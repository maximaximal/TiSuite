Types and strings conversions

tifiles_vartype2string

TIEXPORT2 const char *TICALL tifiles_vartype2string(CalcModel model, uint8_t data)

Returns the type of variable (REAL, EQU, PRGM, ...).

model :
 a calculator model.
data :
 a type ID.
Return value :
 a string like "REAL".
